📊 Bank Loan Portfolio Analysis (Tableau)
🚀 Project Overview

This project focuses on analyzing a bank’s loan portfolio using Tableau to uncover insights related to loan performance, customer behavior, and risk patterns. The goal is to help financial institutions make data-driven decisions, reduce default risk, and optimize loan distribution strategies.

🎯 Objectives
Analyze overall loan distribution and performance
Identify high-risk customers and loan segments
Understand factors affecting loan approval and defaults
Monitor key financial metrics like funded amount, interest, and repayments
Build an interactive dashboard for decision-making
📂 Dataset Description

The dataset contains information about bank loans, including:

Customer details (age, income, employment status)
Loan details (amount, term, purpose)
Loan status (fully paid, charged off, current)
Interest rates and installments
Credit-related attributes
🛠️ Tools & Technologies
Tableau – Data visualization and dashboard creation
Excel / CSV – Data preprocessing
SQL (optional) – Data extraction and transformation
📊 Key KPIs
Total Loan Applications
Total Funded Amount
Total Received Amount
Average Interest Rate
Loan Default Rate
Debt-to-Income Ratio (DTI)
📈 Dashboard Features
📌 Loan Status Distribution (Good vs Bad Loans)
📌 Monthly Trends of Loan Issuance
📌 State-wise Loan Analysis
📌 Purpose-wise Loan Breakdown
📌 Grade & Sub-grade Risk Analysis
📌 Customer Segmentation based on income & employment
🔍 Key Insights
Higher default rates observed in certain loan grades
Customers with high DTI ratio are more likely to default
Specific loan purposes (like small business loans) carry higher risk
Seasonal trends affect loan issuance patterns
💡 Business Impact
Helps banks identify risky loan segments
Improves credit decision strategies
Enhances portfolio performance monitoring
Supports better customer targeting
